# 🌊 Examen API Canvas - Paisaje Marino Interactivo

Este proyecto consiste en la creación de una interfaz web moderna y minimalista que utiliza la **API de Canvas** de HTML5 para renderizar un paisaje marino dinámico. El desarrollo integra **Bootstrap 5** para el diseño responsivo, **CSS3** para el estilo visual y **JavaScript** para la lógica de dibujo técnica.

## 🎨 Detalles Técnicos del Dibujo

El sistema genera una composición de más de **60 figuras básicas** distribuidas de la siguiente manera:
* **Fondo:** 3 Rectángulos (Cielo, Mar, Arena) y 20 líneas de detalle para el oleaje.
* **Elementos Celestes:** 1 Arco para el sol y 15 arcos distribuidos en 5 nubes minimalistas.
* **Fauna:** 10 líneas que componen el vuelo de 5 aves marinas.
* **Transporte Marítimo:** 3 barcos compuestos por trapecios (cascos), líneas (mástiles) y polígonos (velas).
